# doc-fix-hq

This repository contains the frontend for Doc Fix HQ.

## Structure

- `frontend/`: All application code (moved from root)
	- `frontend/src/`: React + TypeScript source
	- `frontend/public/`: Static assets
	- `frontend/index.html`: App entry HTML

Root keeps common config files like `package.json`, `vite.config.ts`, Tailwind, and tsconfig files.
